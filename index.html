HTML
簡介
一種建立網頁的標準標記語言，用來組織架構並呈現網頁內容的程式語言，常與CSS與JavaScript一起使用在網頁設計上。
基礎程式設計
< !DOCTYPE html >
< html >
< !-- created 2010-01-01 -- >
< head >
< title >sample< /title >
< /head >
< body >
< p >Voluptatem accusantium totam rem aperiam.< /p >
< /body >
< /html >
用來代表文件
告訴瀏覽器說，這是以HTML標記語言所撰
寫的文件， 請用HTML的定義解讀的意思
  < html >=開始的
  < /html >=結束
  兩者中間為網頁其內容。
< !DOCTYPE html >
< html>
< !-- created 2010-01-01 -- >
< head >
< title >sample< /title >
< /head >
< body >
< p>Voluptatem accusantium
totam rem aperiam.< /p >
< /body >
< /html >
     網頁的頭
     通常涵蓋標題以及網頁連結
    網頁的身體
    通常用來涵蓋給使用者看的主要內容
< !DOCTYPE html >
< html >
< !-- created 2010-01-01 -- >
< head >
< title>sample< /title >
< /head >
< body >
< p >Voluptatem accusantium
totam rem aperiam.< /p >
< /body >
< /html >
代表網站標題，可另調整字體大小
    表示一個段落 代替換行起到排版的作用
CSS
簡介
CSS負責靜態地修飾網頁，或者配合各種語言動態地對網頁各元素進行格式化。 CSS的功用主要是針對於網站的排版部分，支援大多數的字型字號樣式， 對網頁中的元素進行位置調整擁有對網頁物件和模型樣式編輯的能力。
